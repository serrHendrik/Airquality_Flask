Hardware set-up

What you're going to need

2 x Base Shield v2
Seeeduino V4.2 (Slave)
Seeeduino GPRS (Master) + SIM card
5 x Grove Universal 4 Pin Cable
Grove GPS
Grove Barometer Sensor (BMP280)
Grove Dust Sensor
Grove Multichannel Gas Sensor

How to set up the microstation

Put a Base shield on top of both the GPRS and the seeeduino 4.2.
Using the Grove 4 Pin cables, you can connect the sensors with the Base shields following the scheme below.

Master (GPRS)
Slave Seeeduino V4.2

A. Grove Barometer Sensor (BMP280)

B. Grove Multichannel Gas Sensor

C. Grove Universal 4 Pin Cable

D. Grove GPS

E. Grove Dust Sensor

Software setup

Code

1. Download source code.

2. Unzip the downloaded file

3. Upload the included GPRS library (Seeeduino_GPRS-master.zip) onto the Arduino IDE

4. Change the code in collect_send.ino as seen below, to your PIN, PUK and your internet APN network

5. Set the Board type to 'Arduino Leonardo' and upload the collect_send.ino onto the master seeeduino GPRS

6. Set the Board type to 'Arduino Uno' and upload the slave.ino onto the slave seeeduino

7. Connect the master (with the SIM card inserted) to a power supply to start sending data

8. Register your new setup and read the instructions below to figure out the IMEI Number

REGISTER >>

Get your IMEI Number

1. Connect your seeeduino GPRS to your Arduino IDE with (!) SIM card inserted

2. Open the Arduino IDE with the GPRS library already installed (as explained above)

3. Go to file>Examples>Seeeduino_GPRS-master and click on Serial Debug

4. Upload to the seeeduino and open the Serial Monitor

5. When the serial monitor is ready, send the following command: AT+CPIN?

6. If the response is: +CPIN:READY go to step 8. Otherwise, go to step 7

7. Check if you have to fill in the SIM's PIN (command: AT+CPIN="insertPIN" note: between " " !) or PUK (command: AT+CPIN="insertPUK","insertPIN") and wait for +CPIN:READY

8. At last, send the command to retrieve the IMEI number: AT+GSN
